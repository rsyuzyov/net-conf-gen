# Memory Index

Последняя оптимизация: 2026-05-14

## Проект

- net-conf-gen — генератор сетевых конфигов (Python)
- Точка входа: `main.py`
- Установка: `install.bat`
- Пример конфига: `config.example.yaml`

## Инфраструктура

- Платформа: Windows
- venv активируется в `install.bat`

## Ограничения

- Без README/CHANGELOG если не просили
- ⚠️ **Репозиторий публичный, и память (`.agents/memory/`) коммитится вместе с кодом** (НЕ в .gitignore). Любая запись в память уходит в публичный git. Запрещено писать: пароли, токены, ключи; реальные IP/хосты/FQDN/логины пользователя; имена доменов клиентов; имена внутренних серверов; пути к чувствительным файлам. Только обезличенные факты, рецепты и грабли. Перед каждой записью в diary/topics/MEMORY.md мысленно проверять: «это нормально опубликовать?». Если нет — не писать или обезличить.
